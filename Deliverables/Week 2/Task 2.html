2. Write a program to create a recursive function to calculate the sum of numbers from 0 to 15.

In [7]:
def sum_(n):
    if n == 0:
        return 0
    else:
        return n + sum_(n-1)
    
print('The sum of 0 to 15 is:',sum_(15))

The sum of 0 to 15 is: 120
